Bondwell 12/14 patch for KERMTKP.COM

Original target:
  Kaypro serial status/control port: 06H
  Kaypro serial data port:           04H

New target:
  Bondwell SIO0 data port:           40H
  Bondwell SIO0 status/control port: 41H

Patched file offsets:
  0E3C 06 -> 41
  0E47 04 -> 40
  0E50 06 -> 41
  0E94 04 -> 40
  138B 06 -> 41
  1390 04 -> 40
  1A99 06 -> 41
  1AA1 04 -> 40

The Kaypro-specific terminal-display code remains unchanged.
The serial ready-bit tests remain Kaypro-compatible:
  RX ready: status bit 0
  TX ready: status bit 2
These match the assumptions used by the supplied Kaypro binary and should
be verified on the Bondwell hardware.
